# Mandarin Puzzle Edukasi

Mandarin Puzzle Edukasi adalah aplikasi web pembelajaran kosakata bahasa Mandarin berbasis puzzle susun gambar. Aplikasi ini dirancang untuk membantu pengguna belajar Hanzi, pinyin, arti bahasa Indonesia, contoh kalimat, audio pelafalan, serta latihan interaktif berbasis level.

## Fitur Utama

- Belajar kosakata bahasa Mandarin berdasarkan level
- Menampilkan Hanzi, pinyin, arti, dan contoh kalimat
- Audio pelafalan bahasa Mandarin
- Puzzle susun gambar interaktif
- Sistem login dan register sederhana
- Progress belajar tersimpan otomatis di browser
- Poin dan badge pembelajaran
- Empat level pembelajaran:
  - Pemula
  - Dasar
  - Menengah
  - Lanjutan

## Cara Menjalankan

1. Download atau clone repository ini.
2. Pastikan file utama bernama `index.html`.
3. Buka file `index.html` menggunakan browser.
4. Untuk membuat web online, aktifkan GitHub Pages pada repository.

## Cara Upload ke GitHub Pages

1. Buat repository baru di GitHub.
2. Upload file `index.html`.
3. Upload file `README.md` ini sebagai keterangan proyek.
4. Masuk ke menu `Settings`.
5. Pilih `Pages`.
6. Pada bagian `Build and deployment`, pilih:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Klik `Save`.
8. Tunggu beberapa menit sampai link website muncul.

## Catatan

Aplikasi ini dibuat sebagai media pembelajaran kosakata bahasa Mandarin berbasis web. Seluruh tampilan, style, data kosakata, puzzle, dan script utama berada dalam satu file `index.html`, sehingga tidak memerlukan file tambahan seperti CSS, JavaScript, gambar, atau audio terpisah.

## Tujuan Pengembangan

Aplikasi ini dikembangkan untuk mendukung pembelajaran kosakata bahasa Mandarin melalui pendekatan interaktif, visual, audio, dan gamifikasi agar proses belajar menjadi lebih menarik dan mudah dipahami.

## Lisensi

Proyek ini digunakan untuk keperluan pembelajaran dan penelitian.
